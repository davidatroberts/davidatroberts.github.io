Refreshing the Blog and Moving Some Old Cotent

Apr 5, 2019 #Hugo #Blog

I’ve recently moved the blog from using a really old version of Jekyll to Hugo. I wass really just looking for something a bit more lightweight and my old blog was failing to build on a new version of Jekyll anyway.

You may also notice that I’ve imported a lot of old content from a wordpress blog that I used to have before this one. I’ve took a bit of time to fix some old links to code examples and such that were broken.

Also, going back I’ve noticed that so many of my posts end with something like “will look at blank in the next post” and then I never got around to uploading the next post. Whilst I can’t guarantee that I will follow up all of these - I will definitely be coming back to some of them. As well as posting notes on some abondoned code that I’ve recently dug up.
